Es un proyecto de netbeans, se puede abrir con el IDE, o ejecutar el .jar en la caerpeta dist con el comando:

> java -jar Abb.jar [semilla] [cantidad de numeros para crear] [cantidad de insersiones] [cantidad de busquedas] [cantidad de iteraciones]

o compilar el .java que se encuentra en src/Abb con:

> javac Nodo.java
> javac Abb.java

y ejecutar con:

> java Abb.jar [semilla] [cantidad de numeros para crear] [cantidad de insersiones] [cantidad de busquedas] [cantidad de iteraciones]
